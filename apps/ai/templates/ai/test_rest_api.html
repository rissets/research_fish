API Documentation

            

            
            

                

                    
                    
Available REST API Endpoints

                

                

                    

                        

                            
Base URL: /api/

                        

                        
                        

                            
Session Management:

                            

                                
POST   /sessions/                    - Create new session

                                
GET    /sessions/                    - Get all sessions

                                
GET    /sessions/{id}/               - Get session details

                                
GET    /sessions/{id}/detections/    - Get session detections

                                
POST   /sessions/{id}/close/         - Close session

                            

                        

                        
                        

                            
Prediction APIs:

                            

                                
POST   /predict/                     - Fish segmentation analysis

                                
GET    /predict/model_info/          - Get segmentation model info

                                
POST   /detect/                      - Fish object detection

                                
GET    /detect/model_info/           - Get detection model info

                            

                        

                        
                        

                            
Data APIs:

                            

                                
GET    /detections/                  - Get all detections (filterable)

                                
GET    /detections/{id}/             - Get detection details

                                
GET    /detections/{id}/detected_objects/ - Get objects for detection

                                
GET    /objects/                     - Get detected objects (filterable)

                                
GET    /objects/{id}/                - Get object details

                                
GET    /objects/statistics/          - Get detection statistics

                            

                        

                        
                        

                            
Query Parameters:

                            

                                
- session_id: Filter by session

                                
- prediction: Filter by prediction class
